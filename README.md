# AI Ticket Assistant

## Overview
This project demonstrates how AI (Claude) can be used to summarise and prioritise IT support tickets.

## Problem
Manual triage of tickets is time-consuming and inconsistent.

## Solution
Used Claude prompts to:
- Summarise tickets
- Identify urgency
- Suggest next actions
